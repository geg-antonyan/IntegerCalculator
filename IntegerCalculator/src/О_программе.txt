Калькулятор выполнящие опреации по обратной полской нотации с типом Integer
1.Integer (Integer.h, Integer.cpp) который хранит числа в массиве int поразрядоно. 
Выполняя операции + - / * %  и перевод межу системами счислении (2 <= CC <= 16). 
Перегружены операции сравнении. При попытке операции между чисел с разними СС происходит
бросании в исключении. в программие число разрядов ограничвается 64. все неиспользованные
биты запрлнятся нульями а актулаьное число разрядов хранится в length. Знак числа хранится в
отдельном пкркмнном Sig. Все операции + - / * % выполятся поразрядно используя обычный алгоритм
знакомой всем еще со школы(может присутсвовать багы, но я постарался по возможности исключить их).
2. Контейнеры. Шаблонные классы List (List.h) и Stack(Stack.h)

Логика Программы. Используется паттерн Obsever(Observed.h, Obsever.h, EventData.h) и
все издатели событии защищенно пнаследуются от Observed а для для каждого наблюдателя
создается класс наследовнный публично от Observer. 
 Используется также праттерн command(Command.h) где абстрактный класс Command имет 
не виртульные публичные методы и приватные уже чисто виртуальные методы иполнители котрые 
определябтся в наследниках. Каждая камманда имеет вщзможность отменить свое действие(undo).
  CommandRepository(CommandRepository.h)
Хранти в себя комманды(название и укаэатель на саму комманду) которые можно регистрировать в Command.cpp
(RegisterCommands) и так как у каждой команды есть метод clone  который возврошает копию 
команды то кажды раз при выполнении комады можно взять крпию из хранилище комманд.
	CommandManager(CommandManager.h) как и положена выполняет упровлении команд выполняя
и отменяя комманды так как хранит все выполненные и отменненые команды.
	CommandDispatcher(CommandDispatcher.h) как уже видна из названии выполняет проврку введнной
комманды и сответсвенно решает кому отправлять эти комманды для выполнении
    UserInterface это абстрактый класс который региструрет событие ввода команды и имеет
два метода чисто виртуальные posMassege и dataChanged. от класс наследуется Cli (Command
Line Interface котрый реализует эти меттоды для коммандной строки.
